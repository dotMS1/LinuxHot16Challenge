# HOT 16

# 1. cd
# nie odtwarza płyt ;) cd + nazwa folderu to hasło klucz jak chce się wejśc w jakiś folder.  

# 2. dir
# znam to z DOSa, robi to samo co ls tylko foldery pokazuje na biało

# 3. ls
# robi to samo co dir, tylko foldery pokazuje na niebiesko

# 4. cat> 
# + nazwa pliku i roszerzenie - tworzenie pliku oraz wpisanie od razu treści. HGW co zrobi jeśli damy rozszerzenie .exe ;) 

# 5. mkdir 
# + nazwa folderu - zrobi folder

# 6. rmdir 
# + nazwa folderu - usunie folder

# 7. touch 
# + nazwa plik i rozszerzenie - tworzenie pliku

# 8. nano 
# + nazwa plik i rozszerzenie - stworzy plik i zaczniemy edycje 

# 9. ls -l
# zobaczymy co jest w folderze wraz datami etc

# 10. mk 
# + nazwa pliku - tworzy plik 

# 11. cd ..
# wychodzimy z folderu

# 12. cd //
# odkryte w wyniku literówki. Wchodzimy jednym ruchem do tajemnego katalogu //

# 13. pwd
# ścieżka obecnej lokalizacji

# 14. cp
# kopiowanie pliku np. + nazwa1 nazwa2 - skopiuje plik nazwa1 do nazwa2  

# 15. find
# poszukiwanie pliku po nazwie

# 16. exit
# czyli dobranoc, do widzenia, papa
